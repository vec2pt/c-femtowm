# femtoWM

femtoWM is a [TinyWM](https://github.com/mackstann/tinywm) clone implemented with [XCB](https://xcb.freedesktop.org/). It includes all of TinyWM's features, and the code is structured in a similar way. Just like TinyWM, the main purpose of femtoWM is to serve as a basic example of window manager programming using XCB.

## Controls

- `Alt` + `Button1` + drag - Interactive window move
- `Alt` + `Button3` + drag - Interactive window resize
- `Alt` + `F1`- Raise focused window

## Build from source

To compile and run femtoWM, you need to install the `libxcb` and `libxcb-keysyms` libraries.

1. Clone the repository.
2. Build from source code using the `make` command.
